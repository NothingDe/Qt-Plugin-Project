在windows平台基于VS2017嵌套Qt插件开发的一款桌面应用程序，程序整体架构采用插件框架，各摸块通过插件管理器与主系统进行通讯，主系统主要功能包含xmpp、Mqtt即时通讯，opencv视觉处理，vtk3D点云显示，WebEngine展示WebGL或三维页面，线性动画等
